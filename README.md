# portfolio-assistant

> AI-powered portfolio assistant — ask questions about the brianpelow engineering portfolio.

**Live at**: https://brianpelow.github.io/portfolio-assistant

## Overview

A Claude-powered chatbot that knows everything about the brianpelow engineering
portfolio. Ask it how repos work, how they connect, what problems they solve,
or how the platform architecture is designed.

## How it works

Single-file HTML/JS app deployed on GitHub Pages. Calls the Anthropic API
directly from the browser using your own API key — no backend, no server,
no data stored anywhere.

## Usage

1. Visit the live URL
2. Enter your Anthropic API key in the sidebar
3. Ask anything about the portfolio
